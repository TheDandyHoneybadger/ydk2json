Como hospedar no GitHub Pages

      
    
        
  1. Crie um repositório no GitHub.
    2. 
        
  2. Envie estes arquivos para a raiz do repositório: index.html, styles.css e app.js.
    3. 
        
  3. No GitHub, abra Settings → Pages.
    4. 
        
  4. Em Build and deployment, selecione Deploy from a branch, branch main e pasta /(root).
    5. 
      

      
O app roda 100% no navegador. Ele usa a YGOPRODECK API apenas para resolver nome, tipo e konami_id das cartas.
